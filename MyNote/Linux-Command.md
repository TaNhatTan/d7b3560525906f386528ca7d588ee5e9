#### I. SIMPLE COMMAND
##### 1. Check error code:
```
$ apt-get install mysql-server
or
$ yum install mysql-server
----
$ perror <number>
```

##### 2. Show bandwidth:
```
$ apt-get install vnstat
or
$ yum install vnstat
-----
$ vnstat -i <interface> -l
```
